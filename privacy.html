Política de Privacidad de OceanPlay

Última actualización: 2026-03-05

OceanPlay es una aplicación reproductora multimedia. No proporciona contenido ni listas de reproducción. Cada usuario debe cargar sus propias playlists.

Datos que recopilamos

  • Datos de cuenta / playlist: Si agregas una playlist (servidor, usuario y contraseña), esa información se guarda localmente en tu dispositivo para permitir la reproducción.
  • Diagnóstico (opcional): Si el usuario lo habilita o si se requiere para soporte, se pueden compartir registros básicos para resolver problemas.

Si aún no usas analíticas o reporte de fallos, esta redacción mantiene la política alineada.

Cómo usamos los datos

  • Para cargar y reproducir las playlists que proporcionas.
  • Para mejorar la estabilidad y el rendimiento de la app.
  • Para responder solicitudes de soporte.

Compartición de datos

No vendemos tus datos personales. No compartimos playlists ni credenciales con terceros, salvo cuando tú envías información explícitamente a soporte para diagnóstico.

Almacenamiento de datos

La información de playlists se almacena localmente en tu Apple TV. Puedes eliminarla en cualquier momento borrando la playlist desde la app.
